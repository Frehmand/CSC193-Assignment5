This project is a small JavaScript exercise that adds interactive text-styling features to a webpage. The page includes a textbox and several UI controls that let the user modify the text using JavaScript events.

Features
1. Bigger! Button

Increases the text size in the textarea to 24pt

Shows an alert when clicked

2. FancyShmancy / BoringBetty Radio Buttons

When "FancyShmancy" is selected, the textbox becomes:

Bold

Blue

Underlined

When "BoringBetty" is selected, the textbox returns to:

Normal font weight

Black text

No underline

3. Moo Button

Converts all text in the textarea to uppercase

Adds "-Moo" to the last word of each sentence

Uses JavaScript split() and join() string methods

Files Included

fancifymytext.html

fancifymytext.js

How to Use

Download or clone this repository

Open the file "fancifymytext.html" in any web browser

Type text into the textarea

Use the provided buttons and radio options to apply styles
